About Me

     สวัสดีครับ ผมชื่อนายอนุสรณ์ แปลกจังหรีด เรียนวิศวกรรมคอมพิวเตอร์ ชั้นปีที่ 3 ที่มหาวิทยาลัยขอนแก่น ชอบความท้าทาย อยากจะเรียนรู้สิ่งใหม่ๆ ไม่ว่าจะเป็นทางสาย Programming , Design Art รวมไปถึงด้านเศรษฐศาสตร์ และการเงิน ตอนนี้กำลังศึกษาทางด้าน Smart Contact , Blockchain , React Native และ Data Analysis

     มีประสบการณ์ทางด้าน Web Applications , Mobile applications , IoT และ Smart contract นอกจากนี้ยังมีประสบการณ์ด้าน Graphic design , Video Editor , Animation และ VFX ปัจจุบันกำลังดูแลโปรเจคดังนี้

www.bemoregift.com   ( ได้รับทุน 1 แสนบาทในการพัฒนา )

     bemoregift.com เป็นโครงการรวบรวมร้านขายของฝากในภาคอีสานตอนบน เพื่อสนับสนุนการท่องเที่ยว นวัตกรรม และสินค้าประจำท้องถิ่นต่างๆ ซึ่งได้รับทุนจาก "Innovation Hub KKU" จำนวน 100,000 บาท เพื่อพัฒนาระบบ

IoT Hydro-Organics ( พัฒนาไฮโดรโปรนิกส์โดยปลอดสารเคมี 100% )

     พัฒนาระบบไฮโดรโปนิกส์ให้มีประสิทธิภาพสูงขึ้นโดยใช้ IoT ในการดูแลแบบอัตโนมัติ สามารถควบคุมผ่าน Mobile Applications และต้องการยกระดับให้ระบบไฮโดรโปนิกส์กลายเป็นระบบไฮโดรออแกนิกส์ (ปลอดสารเคมี 100%)
